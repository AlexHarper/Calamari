Applying a transform with an absolute path to a target with an relative to extraction path
Given a package which has the structure:
Acme.Core.1.0.0.nupkg
├─sub
| └─web.config
└─web.config
And the following files exist:
c:\
└─transforms
  └─web.mytransform.config
Then the transform c:\transforms\web.mytransform.config => .\sub\web.config will:
 - Apply the transform c:\transforms\web.mytransform.config to file sub\web.config
